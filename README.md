# lein-meta 

A Leiningen plugin for exposing project configuration.

## Usage

Put `[lein-meta "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your
`:user` profile or your project.clj.

For example:

    $ lein meta version
    0.1.0-SNAPSHOT
